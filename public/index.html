Wie?

In 1994 ben ik begonnen als instructeur bij VDAB. Daar waren het voornamelijk MS-Office opleidingen die ik gaf. Ik heb ook het geluk gehad dat ik bij VDAB heb mogen meewerken met het allereerste afstandsleren-project. Het echte IT-werk begon toen ik docent werd bij βeta VZW onder leiding van Prof. Carlos De Backer. We organiseerden toen voornamelijk werkzoekendenopleidingen voor VDAB. In die eerste jaren waren dat de "oudere" programmeertalen en besturingssystemen: Clipper, C, Perl, PHP, MS-DOS...

In 2004 ben ik zelf voorzitter geworden van βeta. Rond die tijd zijn we ook begonnen met IT-opleidingen voor Cevora. En de lijst van opleidingen werd steeds maar groter:

  • Moderne programmeertalen zoals Java, C# en Python
  • Windows applicaties zoals Windows Server, Exchange, MS SQL Server, ISA Server
  • Webtechnoligieën zoals HTML, CSS, JavaScript, Angular, React
  • Databank technologieën zoals Big Data (Hadoop), normalisatie, SQL en Transact-SQL

Al die jaren ben ik, naast de klassikale opleidingen, ook geïnteresseerd gebleven in allerlei technieken rond afstandsleren. In 2014 heb ik voor Cevora een zelfstudiecursus JavaScript ontwikkeld. Maar ik heb mij pas echt kunnen uitleven in online-opleidingen dankzij Corona toen dat de enige mogelijkheid was om opleidingen te geven.

Online opleidingen zijn anders dan klassikale opleidingen. Daarom moeten ze ook anders aangepakt worden.

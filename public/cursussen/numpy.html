Numpy is een Python bibliotheek die gebruikt wordt om op een efficiënte manier berekeningen uit te voeren. Daardoor is deze bibliotheek belangrijk wanneer je met grote hoeveelheden data wil werken, zoals bij een data-analyse.

Je zult Numpy voornamelijk tegenkomen wanneer je Python bibliotheken gebruikt voor data-analyse en machine learning, zoals Pandas en Scikit-learn.

De opleiding wordt online gegeven via Microsoft Teams. Om de opleiding te volgen kun je gebruik maken van Google colab. Wanneer je Python lokaal op je PC kunt installeren, kan dat natuurlijk ook. Je hebt dan nog wel een editor nodig zoals PyCharm of Visual Studio Code.

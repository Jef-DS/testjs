SQL is de taal die we gebruiken om te werken met relationele databanken. Je kunt hiermee niet alleen gegevens opvragen, maar ook wijzigen. Er bestaat een SQL-standaardtaal, maar elke databank heeft zijn eigen SQL dialect dat licht afwijkt van de standaard.

Tijdens de opleiding maken we gebruik van de databank PostgreSQL. Dat is de databank die de standaard het beste volgt. Wanneer de belangrijkste databanken (Oracle, MS SQL Server, MySQL, MariaDB) hiervan afwijken, wordt dat kort vermeld.

De opleiding wordt online gegeven via Microsoft Teams. Om de opleiding te kunnen volgen, moet je geen extra software installeren. De PostgreSQL databank is toegankelijk via een web interface.

Programma

  • Vooraf: Wat is een relationele databank?
  • Sessie 1: Gegevens opvragen uit 1 tabel, sorteren en filteren (SELECT FROM, WHERE, ORDER BY)
  • Sessie 2: Aggregatiefuncties(COUNT, MIN, MAX, SUM, AVG) en groeperen van data (GROUP BY)
  • Sessie 3: Gegevens opvragen uit meerdere tabellen (INNER/LEFT/RIGHT/OUTER JOIN)
  • Sessie 4: Subqueries en gegevens aanpassen (INSERT, UPDATE, DELETE)
